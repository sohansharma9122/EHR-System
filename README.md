🏥 Electronic Health Record (EHR) System
A secure and scalable Electronic Health Record (EHR) web application built using Django and Django REST Framework.
This system helps manage patients, doctors, staff, and medical records efficiently in a centralized platform.

🚀 Features
---------------

👨‍⚕️ Doctor Management
🧑‍🤝‍🧑 Patient Management
🏢 Staff Management
🔐 Authentication (Login/Register/Logout)
📁 Medical Record Management
🛡️ Secure Admin Panel
RESTful API integration

🛠️ Tech Stack
-----------------
Python
Django
Django REST Framework
HTML
CSS
SQLite (Default Database)
